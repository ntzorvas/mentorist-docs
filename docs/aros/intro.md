# AROS

**AROS** (Autonomous Restaurant Operating System) is an AI-driven platform for restaurant automation and management.

## Overview

AROS enables restaurants to operate with minimal manual intervention through:

- **Inventory Management** — automated stock tracking and reorder triggers
- **Order Processing** — AI-assisted order handling and routing
- **Social Media** — automated content generation via the Vaggelis agent
- **Security Monitoring** — real-time anomaly detection

## Architecture

AROS is built on:

- TypeScript + Node.js backend
- Anthropic Claude SDK for AI agents
- BullMQ for task scheduling
- Multi-agent system with specialized roles

## Agents

| Agent | Role |
|---|---|
| InventoryAgent | Stock levels, reorder alerts |
| OrderAgent | Order lifecycle management |
| SecurityAgent | Threat detection, access control |
| ClaudeExpertAgent | General-purpose AI assistant |
| Vaggelis | Social media content creation |
