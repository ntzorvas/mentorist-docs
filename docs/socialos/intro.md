# SocialOS

**SocialOS** is an autonomous social media management platform for multi-brand content creation and scheduling.

## Overview

SocialOS provides end-to-end social media automation:

- **Content Generation** — AI-written posts tailored per brand voice
- **Multi-platform Publishing** — Instagram, Facebook, TikTok, and more
- **Scheduling** — automated calendar with peak-time optimization
- **Analytics** — engagement tracking and performance insights

## Key Features

| Feature | Description |
|---|---|
| Brand Profiles | Per-restaurant voice and style configuration |
| Content Queue | Scheduled posts with approval workflow |
| AI Copywriting | Claude-powered captions and hashtags |
| Media Assets | Image and video management |
