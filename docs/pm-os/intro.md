# Claude PM-OS

**Claude PM-OS** is an AI-powered project management operating system that orchestrates tasks, runs, and agents autonomously.

## Overview

PM-OS provides a full task lifecycle management system with:

- **AutoScheduler** — automatically picks up pending tasks and executes them
- **Agent Runs** — each task runs as an isolated Claude Code session
- **Built-in Tools** — `write_file`, `read_file`, `list_directory`, `run_bash`
- **Dashboard** — real-time status of projects, tasks, and agents

## Key Concepts

| Concept | Description |
|---|---|
| Project | A collection of related tasks |
| Task | A unit of work with acceptance criteria |
| Run | An execution instance of a task |
| Agent | A specialized Claude instance for a domain |
